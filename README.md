# Abhishek B K — Portfolio

A single-page personal portfolio styled as a retro **UEFI/BIOS setup utility** — built with plain HTML, CSS, and vanilla JS. No frameworks, no build step.

**Live site:** __

## Preview

Boots up with a simulated system-loading sequence, then reveals a BIOS-style setup screen with keyboard-menu-inspired navigation across five tabs: Overview, Hardware (skills), Boot Log (projects), System Info (education/certifications), and Contact.

## Tech

- HTML5 + CSS3 (custom properties, no framework)
- Vanilla JavaScript (tab switching, boot animation, live clock)
- Font: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Run locally

Just open `index.html` in a browser — no install, no server needed.

```bash
https://github.com/zei-abhi-test/Portfolio.git
cd <repo-name>
open index.html   # or double-click the file
```

## Deploy

Deployed via [Netlify](https://netlify.com):

1. Push this repo to GitHub
2. Netlify → **Add new site → Import an existing project**
3. Connect the repo, leave build settings empty, publish directory `/`
4. Deploy
